# Стандарты 1С: Именование полей и алиасов в запросах

**Источник:** [ITS #std437](https://its.1c.ru/db/v8std/content/437/hdoc), [ITS #std758](https://its.1c.ru/db/v8std/content/758/hdoc), [v8std.ru](https://v8std.ru)  
**Связанные инструменты:** `analyze_queries`, `validate_query_static`, `generate_query`

---

## Правила именования алиасов источников данных (КАК)

### #std758: Псевдонимы источников данных

1. **ВСЕ источники данных** должны иметь алиас через `КАК`
2. Алиас должен быть **коротким** (1-10 символов) и **осмысленным**
3. Для основной таблицы — 1-буквенный алиас: `Т`, `А`, `Б`
4. Для JOIN'ов — осмысленное имя: `Контрагенты`, `Договоры`
5. Не использовать зарезервированные слова как алиас
6. Обращение к полям **только через алиас**: `Т.Поле`, не `Поле`

### Примеры

```bsl
// ПРАВИЛЬНО
"ВЫБРАТЬ
|    Рег.Номенклатура КАК Номенклатура,
|    Рег.СуммаВыручки КАК Выручка
|ИЗ
|    РегистрНакопления.Продажи КАК Рег
|    ЛЕВОЕ СОЕДИНЕНИЕ Справочник.Контрагенты КАК Контрагенты
|        ПО Рег.Контрагент = Контрагенты.Ссылка"

// ПЛОХО: нет алиаса
"ВЫБРАТЬ Номенклатура, СуммаВыручки ИЗ РегистрНакопления.Продажи"

// ПЛОХО: обращение без алиаса
"ВЫБРАТЬ Номенклатура ИЗ РегистрНакопления.Продажи КАК Рег"
```

## Правила именования полей в SELECT (КАК)

1. **ВЫЧИСЛЯЕМЫЕ поля** — алиас обязателен: `СУММА(Т.Сумма) КАК Итог`
2. **АГРЕГАТНЫЕ функции** — алиас обязателен: `СУММА(Т.Сумма) КАК Сумма`
3. **Поля источника** — алиас желателен: `Т.Номенклатура КАК Номенклатура`
4. **Алиас** — существительное в именительном падеже
5. **Без сокращений** — `Номенклатура`, не `Ном`; `Контрагент`, не `Контр`
6. **Англоязычный алиас** — только если в конфигурации используется английский язык
7. **Алиас вычисляемого поля** — описывает результат: `Прибыль`, `СуммаДолга`, `КоличествоДней`

### Шаблоны именования

| Тип поля | Паттерн | Пример |
|----------|---------|--------|
| Прямое поле | `Т.Поле КАК Поле` | `Рег.Номенклатура КАК Номенклатура` |
| Агрегат | `ФУНКЦИЯ(Т.Поле) КАК ИмяРезультата` | `СУММА(Рег.Сумма) КАК Сумма` |
| Вычисляемое | `Т.Поле1 - Т.Поле2 КАК ИмяРезультата` | `Рег.Выручка - Рег.Себестоимость КАК Прибыль` |
| Выражение | `ВЫРАЗИТЬ(Т.Поле КАК Тип) КАК Поле` | `ВЫРАЗИТЬ(Рег.Регистратор КАК Документ.Заказ) КАК Регистратор` |
| Подзапрос | `(ВЫБРАТЬ ...) КАК ИмяПоля` | `(ВЫБРАТЬ К.ИНН ИЗ ...) КАК ИНН` |
