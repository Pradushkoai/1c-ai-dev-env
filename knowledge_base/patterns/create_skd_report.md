# Паттерн: Создание отчёта на СКД

## Когда использовать
Когда нужно создать отчёт в 1С 8.3. СКД (Схема Компоновки Данных) — основной механизм отчётов.

## Структура

```
Reports/
└── ИмяОтчета/
    ├── ИмяОтчета.xml                    # Метаданные
    ├── Ext/
    │   └── Module.bsl                   # Модуль объекта (СКД-логика)
    ├── Templates/
    │   └── ОсновнаяСхемаКомпоновкиДанных/
    │       ├── ОсновнаяСхемаКомпоновкиДанных.xml
    │       └── Ext/
    │           └── Template.xml         # СКД-схема
    └── Forms/
        └── ФормаОтчета/
            ├── ФормаОтчета.xml
            └── Ext/Form/
                ├── Form.xml
                └── Module.bsl           # Модуль формы
```

## СКД-схема (Template.xml)

```xml
<DataCompositionSchema xmlns="http://v8.1c.ru/8.1/data-composition-system/schema">
    <dataSource>
        <name>ИсточникДанных1</name>
        <dataSourceType>Local</dataSourceType>
    </dataSource>
    <dataSet xsi:type="DataSetQuery">
        <name>НаборДанных1</name>
        <field xsi:type="DataSetFieldField">
            <dataPath>Контрагент</dataPath>
            <field>Контрагент</field>
        </field>
        <field xsi:type="DataSetFieldField">
            <dataPath>Сумма</dataPath>
            <field>Сумма</field>
        </field>
        <dataSource>ИсточникДанных1</dataSource>
        <query>
            ВЫБРАТЬ
                РеализацияТоваровУслуг.Контрагент КАК Контрагент,
                РеализацияТоваровУслуг.СуммаДокумента КАК Сумма
            ИЗ
                Документ.РеализацияТоваровУслуг КАК РеализацияТоваровУслуг
            ГДЕ
                РеализацияТоваровУслуг.Дата МЕЖДУ &amp;ПериодНачало И &amp;ПериодКонец
        </query>
    </dataSet>
    <parameter>
        <name>ПериодНачало</name>
        <title>
            <v8:item>
                <v8:lang>ru</v8:lang>
                <v8:content>Период с</v8:content>
            </v8:item>
        </title>
        <valueType>
            <v8:Type>xs:dateTime</v8:Type>
            <v8:DateQualifiers>
                <v8:DateFractions>DateTime</v8:DateFractions>
            </v8:DateQualifiers>
        </valueType>
    </parameter>
</DataCompositionSchema>
```

## Модуль объекта (Module.bsl)

```bsl
#Область ПрограммныйИнтерфейс

Функция СформироватьОтчет(ПараметрыОтчета = Неопределено) Экспорт
    
    // Получаем СКД-схему из макета
    СхемаКомпоновкиДанных = ПолучитьМакет("ОсновнаяСхемаКомпоновкиДанных");
    
    // Настройки компоновки
    НастройкиКомпоновки = СхемаКомпоновкиДанных.НастройкиПоУмолчанию;
    
    // Установка параметров
    Если ПараметрыОтчета <> Неопределено Тогда
        Для Каждого КлючЗначение Из ПараметрыОтчета Цикл
            ПараметрСКД = НастройкиКомпоновки.ПараметрыДанных.НайтиЗначениеПараметра(
                Новый ПараметрКомпоновкиДанных(КлючЗначение.Ключ));
            Если ПараметрСКД <> Неопределено Тогда
                ПараметрСКД.Значение = КлючЗначение.Значение;
                ПараметрСКД.Использование = Истина;
            КонецЕсли;
        КонецЦикла;
    КонецЕсли;
    
    // Компоновка данных
    КомпоновщикМакета = Новый КомпоновщикМакетаКомпоновкиДанных;
    МакетКомпоновки = КомпоновщикМакета.Выполнить(СхемаКомпоновкиДанных, НастройкиКомпоновки);
    
    // Выполнение
    ПроцессорКомпоновки = Новый ПроцессорКомпоновкиДанных;
    ПроцессорКомпоновки.Инициализировать(МакетКомпоновки);
    
    // Вывод
    Результат = Новый ТабличныйДокумент;
    ПроцессорВывода = Новый ПроцессорВыводаРезультатаКомпоновкиДанныхВТабличныйДокумент;
    ПроцессорВывода.УстановитьДокумент(Результат);
    ПроцессорВывода.Вывести(ПроцессорКомпоновки);
    
    Возврат Результат;
    
КонецФункции

#КонецОбласти
```

## Best Practices

1. **Параметры**: всегда используйте параметры вместо жёстко заданных значений
2. **Отборы**: используйте отборы СКД, не пишите WHERE в запросе
3. **Группировки**: используйте группировки СКД вместо GROUP BY в запросе
4. **Условное оформление**: форматируйте результат через СКД, не в коде
5. **Настройки по умолчанию**: определите в СКД, не в коде
6. **Форма отчёта**: используйте стандартную форму с компоновщиком настроек

## Антипаттерны

- ❌ Формирование отчёта через запрос + табличный документ (используйте СКД)
- ❌ Жёстко заданные даты в запросе (используйте параметры)
- ❌ GROUP BY в запросе (используйте группировки СКД)
- ❌ Программное форматирование ячеек (используйте условное оформление СКД)
