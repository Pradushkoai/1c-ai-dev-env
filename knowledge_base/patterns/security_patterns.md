```bsl

        "ВЫБРАТЬ

    Возврат Запрос.Выполнить().Выгрузить();
```

## Поиск по коду с проверкой

```bsl
// ПРАВИЛЬНО: проверка результата
Ссылка = Справочники.Номенклатура.НайтиПоКоду("00001");
Если Ссылка.Пустая() Тогда
    ВызватьИсключение "Элемент не найден";
КонецЕсли;

// ПЛОХО: без проверки
Ссылка = Справочники.Номенклатура.НайтиПоКоду("00001");
Объект = Ссылка.ПолучитьОбъект(); // упадёт если пустая
```

## Правила

1. **ВСЕГДА** использовать параметры запроса (`&Параметр`)
2. **НИКОГДА** не конкатенировать строки в тексте запроса
3. **Безопасный режим** — не отключать без необходимости
4. **Привилегированный режим** — только в модулях с `ПолныеПрава`
5. **Проверка результатов** — всегда проверять `НайтиПоКоду`, `НайтиПоНаименованию`
6. **Выполнить()** — использовать только с безопасным режимом
7. **COM-объекты** — только в безопасном контексте
