# Антипаттерны 1С разработки

## 🔴 Критичные

### 1. Бизнес-логика в модуле формы
**Плохо:**
```bsl
// Form/Module.bsl
&НаСервере
Процедура ВыполнитьРасчетНаСервере()
    Запрос = Новый Запрос;
    Запрос.Текст = "ВЫБРАТЬ ...";
    // расчёты
КонецПроцедуры
```

**Хорошо:**
```bsl
// Object/Module.bsl
Функция ВыполнитьРасчет() Экспорт
    // расчёты
КонецФункции

// Form/Module.bsl
&НаСервере
Процедура ВыполнитьРасчетНаСервере()
    Результат = РеквизитФормыВЗначение("Объект").ВыполнитьРасчет();
КонецПроцедуры
```

### 2. Запросы в цикле
**Плохо:**
```bsl
Для Каждого Элемент Из Список Цикл
    Запрос = Новый Запрос;
    Запрос.Текст = "ВЫБРАТЬ * ИЗ Справочник.Номенклатура ГДЕ Ссылка = &Ссылка";
    Запрос.УстановитьПараметр("Ссылка", Элемент);
    Результат = Запрос.Выполнить();
КонецЦикла;
```

**Хорошо:**
```bsl
Запрос = Новый Запрос;
Запрос.Текст = "ВЫБРАТЬ * ИЗ Справочник.Номенклатура ГДЕ Ссылка В (&Список)";
Запрос.УстановитьПараметр("Список", Список);
Результат = Запрос.Выполнить();
```

### 3. Отсутствие обработки ошибок
**Плохо:**
```bsl
Файл = Новый Файл(Путь);
Текст = Файл.Прочитать();
```

**Хорошо:**
```bsl
Попытка
    Файл = Новый Файл(Путь);
    Текст = Файл.Прочитать();
Исключение
    ЗаписьЖурналаРегистрации("Ошибка чтения файла", УровеньЖурналаРегистрации.Ошибка,,,
        ОписаниеОшибки());
    ВызватьИсключение "Не удалось прочитать файл: " + Путь;
КонецПопытки;
```

### 4. Прямой доступ к БД из формы
**Плохо:**
```bsl
&НаКлиенте
Процедура ПриИзменении(Элемент)
    Запрос = Новый Запрос; // ❌ Нельзя на клиенте!
КонецПроцедуры
```

**Хорошо:**
```bsl
&НаКлиенте
Процедура ПриИзменении(Элемент)
    Данные = ПолучитьДанныеНаСервере();
КонецПроцедуры

&НаСервере
Функция ПолучитьДанныеНаСервере()
    Запрос = Новый Запрос;
    // ...
КонецФункции
```

## 🟡 Важные

### 5. Неправильное именование
**Плохо:** `Процедура р1()`, `Перем а`
**Хорошо:** `Процедура ВыполнитьРасчет()`, `Перем СуммаИтога`

### 6. Отсутствие комментариев
**Плохо:**
```bsl
Функция Расч(А, Б)
    Возврат А * Б * 1.2;
КонецФункции
```

**Хорошо:**
```bsl
// Рассчитывает сумму с НДС 20%
//
// Параметры:
//  Сумма - Число - сумма без НДС
//  Ставка - Число - ставка НДС (0.20)
//
// Возвращаемое значение:
//  Число - сумма с НДС
Функция РассчитатьСуммуСНДС(Сумма, Ставка = 0.20) Экспорт
    Возврат Сумма * (1 + Ставка);
КонецФункции
```

### 7. Магические числа
**Плохо:**
```bsl
Сумма = Количество * 1000 * 1.2;
```

**Хорошо:**
```bsl
Перем КонстантаЦены, СтавкаНДС;
Сумма = Количество * КонстантаЦены * (1 + СтавкаНДС);
```

### 8. Сложные условия
**Плохо:**
```bsl
Если А > 0 И Б < 10 И В <> 5 И Г = Истина И Д > 100 Тогда
```

**Хорошо:**
```bsl
УсловиеВыполнено = (А > 0) И (Б < 10) И (В <> 5);
Если УсловиеВыполнено И (Г = Истина) И (Д > 100) Тогда
```

## 🟢 Стилистические

### 9. Отсутствие областей
**Плохо:** весь код в одной куче
**Хорошо:**
```bsl
#Область ПрограммныйИнтерфейс
// Экспортные процедуры
#КонецОбласти

#Область СлужебныйПрограммныйИнтерфейс
// Служебные экспортные
#КонецОбласти

#Область СлужебныеПроцедурыИФункции
// Внутренние
#КонецОбласти
```

### 10. Длинные процедуры
**Плохо:** процедура на 200 строк
**Хорошо:** разбить на 5 процедур по 40 строк
