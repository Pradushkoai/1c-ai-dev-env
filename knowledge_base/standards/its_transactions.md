## #std651: Транзакции: правила использования

**ITS:** https://its.1c.ru/db/v8std/content/651/hdoc

### Правила
1. `НачатьТранзакцию()` — ВСЕГДА перед `Попытка`
2. `ЗафиксироватьТранзакцию()` — ВСЕГДА внутри `Попытка`
3. `ОтменитьТранзакцию()` — ВСЕГДА внутри `Исключение`
4. После `ОтменитьТранзакцию` — `ВызватьИсключение` для проброса
5. Минимум кода внутри транзакции — только запись
6. Не делать запросы к БД без необходимости
7. Не вызывать интерактивные операции (Предупреждение, ОткрытьФорму)

### Пример
```bsl
НачатьТранзакцию();
Попытка
    Объект.Записать();
    ЗафиксироватьТранзакцию();
Исключение
    ОтменитьТранзакцию();
    ЗаписьЖурналаРегистрации("Ошибка", УровеньЖурналаРегистрации.Ошибка, , , ОписаниеОшибки());
    ВызватьИсключение;
КонецПопытки;
```
**BSL LS:** `BeginTransactionBeforeTryCatch`, `CommitTransactionOutsideTryCatch`, `PairingBrokenTransaction`

## #std648: Блокировка данных объекта для редактирования из кода

**ITS:** https://its.1c.ru/db/v8std/content/648/hdoc

### Правила
- Использовать `БлокировкаДанных` перед чтением объекта в транзакции
- Блокировать только те объекты, которые будут изменяться
- Не блокировать справочники целиком — только конкретные элементы

## #std649: Чтение отдельных реквизитов объекта из базы данных

**ITS:** https://its.1c.ru/db/v8std/content/649/hdoc

### Правила
- Для чтения отдельных полей использовать запрос, а не `ПолучитьОбъект()`
- `Объект.ПолучитьОбъект()` — загружает ВСЕ данные (медленно)
- Запрос с `ВЫБРАТЬ Поле1, Поле2` — загружает только нужное (быстро)

### Пример
```bsl
// ПЛОХО: загружает весь объект
Объект = Ссылка.ПолучитьОбъект();
ДатаДокумента = Объект.Дата;

// ПРАВИЛЬНО: только нужное поле
Запрос = Новый Запрос("ВЫБРАТЬ Док.Дата ИЗ Документ.Заказ КАК Док ГДЕ Док.Ссылка = &Ссылка");
```

## #std650: Блокирующее чтение остатков в начале транзакции

**ITS:** https://its.1c.ru/db/v8std/content/650/hdoc

### Правила
- При списании/поступлении — блокирующее чтение остатков В НАЧАЛЕ транзакции
- Использовать `Документ.Записать()` с `РежимЗаписиДокумента.Запись` перед изменением регистра
- Не читать остатки без блокировки если будет списание
