# Role-Switching Protocol

## Протокол A: Полный (сложные задачи)
🧠 Архитектор (план) → 👨‍💻 Программист (код) → 🔍 Ревьюер (проверка) → 📝 Документатор (описание + learning)

## Протокол B: Быстрый (простые задачи)
👨‍💻 Программист → 🔍 Ревьюер (быстрая)

## Протокол C: Code Review
🔍 Ревьюер → 🧠 Архитектор → 📝 Документатор
