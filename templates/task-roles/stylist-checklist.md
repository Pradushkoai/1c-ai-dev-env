## Чек-лист Стилиста

### Структура модуля (STD 454)
- [ ] Области расположены в правильном порядке:
  1. ПрограммныйИнтерфейс
  2. СлужебныйПрограммныйИнтерфейс
  3. СлужебныеПроцедурыИФункции
  4. Инициализация
- [ ] Все экспортные процедуры — в ПрограммныйИнтерфейс или СлужебныйПрограммныйИнтерфейс
- [ ] Нет процедур вне областей

### Читаемость
- [ ] Длина строк ≤ 120 символов
- [ ] Вложенность ≤ 3 уровней
- [ ] Имена осмысленные (без `А`, `Б`, `Пер`)
- [ ] Комментарии на русском, код соответствует стилю 1С

### Типографика
- [ ] em-dash (—) вместо дефиса (-) в комментариях
- [ ] Используется «ё» (не «е»)
- [ ] Нет закомментированного кода

### Антипаттерны (запрещено)
- [ ] Нет `Сообщить()` — использовать `ОбщегоНазначения.СообщитьПользователю`
- [ ] Нет `Выполнить()` / `Вычислить()`
- [ ] Нет точечной нотации в запросах (Товар.Цена) — через `Ссылка.Свойство`
- [ ] Нет конкатенации в запросах — `УстановитьПараметр`
- [ ] Нет `Попытка...Исключение` вокруг БД-операций без логирования

### Клиент-сервер (STD 12)
- [ ] `НачатьТранзакцию` только в `&НаСервере`
- [ ] Обращение к БД только в `&НаНаСервере` (не в `&НаКлиенте`)
- [ ] Серверные вызовы в цикле — устранены
- [ ] `ОповеститьОбИзменении` только на клиенте
