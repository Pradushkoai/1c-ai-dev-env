//

//   <Параметр1> - <Тип> - <Назначение>.
//   <Параметр2> - <Тип> - <Назначение>.
//

//   <Тип> - <описание>.

## Примеры использования (snippets)

### Пример 1: Создание нового товара
```bsl
// Создать товар с базовыми атрибутами
Ссылка = СоздатьТовар(
    "Молоток слесарный",
    "M-001",
    Справочники.ЕдиницыИзмерения.Штука
);

// Установить цену
УстановитьЦену(Ссылка, 250.00, ДатаЗаписи);
```

### Пример 2: Поиск товара по артикулу
```bsl
НайденныйТовар = НайтиПоАртикулу("M-001");
Если НайденныйТовар = Справочники.Товары.ПустаяСсылка() Тогда
    ОбщегоНазначения.СообщитьПользователю("Товар не найден");
КонецЕсли;
```
