# Code of Conduct

## Наши принципы

1. **Уважение** — уважай других участников, независимо от уровня
2. **Конструктивность** — критика должна быть полезной, не разрушительной
3. **Честность** — говори правду, даже если она неприятна
4. **Ответственность** — признавай ошибки и исправляй их

## Недопустимое поведение

- Оскорбления и личные нападки
- Дискриминация по любому признаку
- Спам и реклама
- Намеренное введение в заблуждение

## Нарушения

Нарушения могут привести к блокировке. Решение принимает мейнтейнер.
